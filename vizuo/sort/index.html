SORTING VISUALIZER


A web based program that allows users to see how different sorting algorithms work by visually demonstrating the sorting process step by step. Sorting algorithms are commonly used in computer science to arrange data in a particular order, such as ascending or descending. The program is a "bar chart" visualization, where each element in the list is represented by a bar whose height corresponds to the element's value. As the sorting algorithm runs, the bars move and change color to reflect the current state of the list. Click one of the below buttons to observe the data manipulation in action.
